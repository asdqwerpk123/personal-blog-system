# uni-app P3 文档与部署可行计划

## 1. P3 目标

P3 不继续开发业务功能，不修改后端接口、不修改 Vue 后台、不修改 uni-app 功能代码，也不调整数据库结构。P3 的目标是把 uni-app 第一版课程项目闭环所需的运行说明、演示材料、部署说明和验收清单补齐，便于本地演示、H5 发布和小程序域名配置。

本阶段交付物包括：

- 运行与联调说明
- 演示截图清单
- Nginx 反向代理部署说明
- uni-app H5 打包部署说明
- 小程序合法域名配置说明
- 最终验收清单

## 2. 当前功能基线

| 功能项 | 当前基线 |
| --- | --- |
| 后端公开接口 | 已有 `/public/**` 文章、分类、标签、评论、友链接口 |
| 用户登录入口 | 已有 `POST /user/auth/login`，面向普通 `USER` 用户 |
| uni-app 首页 | 已支持公开文章列表、搜索、分类筛选、标签筛选、下拉刷新、上拉加载 |
| 分类与标签 | 已支持分类页、标签页和首页筛选 |
| 文章详情 | 已支持标题、作者、时间、封面、正文、标签展示 |
| 评论 | 已支持文章评论分页展示和登录用户发表评论 |
| 登录注册 | 已支持普通用户登录、注册和登录态保存 |
| 我的页面 | 已支持未登录态、资料加载、资料编辑、退出登录 |
| 头像上传 | 已支持 `uni.uploadFile` 上传头像 |
| 我的评论 | 已支持我的评论分页加载 |
| 请求竞态修复 | 首页文章请求已避免旧请求覆盖新筛选结果 |

## 3. P3 分阶段

| 阶段 | 目标 | 具体任务 | 涉及文档 | 验收标准 | 风险点 |
| --- | --- | --- | --- | --- | --- |
| P3-A | 补齐运行说明 | 说明后端启动、数据库准备、HBuilderX H5 预览、常用接口检查、演示流程 | `docs/uni-app-runbook.md` | 新同学可按文档启动后端并预览 uni-app H5 | 本地 MySQL、Maven 仓库权限、8081 端口占用 |
| P3-B | 固化演示截图清单 | 梳理演示前、核心功能、异常场景截图和命名建议 | `docs/uni-app-demo-checklist.md` | 课程演示材料可按清单逐项截图 | 测试数据不足会影响截图完整性 |
| P3-C | 补齐 Nginx 与 H5 部署文档 | 说明 H5 静态资源托管、`/api/` 代理、`/uploads/` 图片访问、H5 构建发布 | `docs/uni-app-nginx-deploy.md`、`docs/uni-app-h5-build.md` | 能明确生产 `baseURL`、代理关系和刷新回退配置 | 线上域名、HTTPS、图片存储路径不一致 |
| P3-D | 补齐小程序域名说明 | 说明 request、uploadFile、downloadFile、图片域名配置和环境切换 | `docs/uni-app-miniprogram-domain.md` | 小程序调试和正式环境域名要求清晰 | 正式环境不能使用 localhost 或自签名证书 |
| P3-E | 最终验收 | 汇总运行、接口、页面、登录、评论、图片、部署验收项 | 本计划和各专项文档 | P3 文档覆盖第一版演示与部署闭环 | 文档与实际配置漂移，需要随代码变更同步 |
